About

This viewer provides three different camera stream modes:

  • Regular Stream: Raw camera feed without any processing
  • Face Detection: Detects and highlights faces in the stream
  • Face Recognition: Identifies known faces and labels them

To add a new person, click the edit icon on an unnamed face and enter a name for that person.

Click "Drag to Merge" and use drag and drop to easily merge similar faces.
